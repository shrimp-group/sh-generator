- **分层架构**：采用标准的 Controller-Service-Mapper 三层结构，REST 接口定义在 `rest` 包，业务逻辑在 `service` 包，数据访问在 `mapper` 包。
- **自动装配**：通过 `GeneratorServerConfig` 配合 `META-INF/spring/org.springframework.boot.autoconfigure.AutoConfiguration.imports` 实现 Spring Boot Starter 式的自动配置与组件扫描。
- **动态数据源**：利用 `DynamicDataSourceInit` 实现运行时根据配置动态创建数据库连接，支持多租户或多库元数据读取。
- **路由管理**：使用 `Route` 接口集中定义 API 路径常量，并通过自定义 `@Router` 注解进行模块化管理。
- **代码生成引擎**：核心逻辑位于 `GenService`，结合 FreeMarker 模板引擎与数据库元数据（TableInfo/ColumnInfo）动态渲染并打包代码文件。
