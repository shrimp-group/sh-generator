- **技术栈**: 采用 Vue 3 (Composition API) + Vite 构建，状态管理使用 Pinia，UI 框架为 Element Plus。
- **目录结构**: 遵循 RuoYi-Vue3 规范，`src/api` 封装后端接口，`src/views` 存放业务页面（如代码生成、系统配置），`src/components` 包含通用组件（如 Crontab、MonacoEditor）。
- **核心逻辑**: `src/utils/generator` 实现了可视化表单到 Vue 代码的动态渲染引擎；`src/permission.js` 结合 Pinia 实现基于角色的动态路由加载与权限控制。
- **交互模式**: 通过 Axios 拦截器统一处理 Token 认证、重复提交校验及全局异常提示。
