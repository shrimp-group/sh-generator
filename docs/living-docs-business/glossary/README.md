## 术语分类

### 按领域分类

- **通用术语** - 各业务线通用（动态数据源、FK、PK）
- **领域术语** - 代码生成器专用（生成项目、生成任务、生成模板、目标数据源、生成日志等）

### 术语命名规范

- 术语名采用中文，明确简洁
- 英文名采用标准英文翻译
- 缩写采用业界通用形式
