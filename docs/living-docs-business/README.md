## 业务领域概述

sh-generator 是一个面向研发团队的代码生成器。其核心业务价值是：用户通过 Web 控制台或 Maven 插件配置"生成项目 → 目标数据源 → 表前缀 → 生成任务 → 模板"五要素后，系统自动读取目标数据库的表与列元数据，按 FreeMarker 模板渲染出符合团队规范的工程代码（Entity / Mapper / Service / Rest 等），并以 zip 形式下载或集成进构建流程，从而显著降低重复性脚手架代码的编写成本，统一团队代码风格。

业务边界：

- 只做"按模板生成代码"，不做模型驱动开发（MDD）或复杂业务逻辑生成。
- 仅读取目标库的 `information_schema` 元数据，不读取/修改业务数据。
- 生成的代码产物落盘为临时文件并打包 zip，不直接回写用户业务库。
- 不负责代码的编译、部署、运行，仅产出源码。
