## critical 级别债务特别说明

> **TD-001 路径逃逸防护薄弱** 为当前唯一 critical 级别债务。由于 `projectBasePath` 来自用户可配置的 `gen_task.project_base_path`，而现有 `replace("../", "parent/")` 仅做字面量替换，无法防御 `..\`、`..%2F` 等绕过，可能导致任意文件写入。**建议在生产部署前优先修复**，修复方案见 [安全风险/001-路径逃逸防护薄弱.md](安全风险/001-路径逃逸防护薄弱.md)。

## 扫描范围与依据

本次扫描基于 `generator-server` 模块的全部源码与 `generator-server-starter` 的 `application.yml`，重点关注：

- `GenService` 核心生成引擎（路径处理、文件 IO、数据源切换、异常处理）
- `GenDatasource` 密码存储与脱敏
- `application.yml` 运维端点配置

未覆盖 `generator-client` 与 `generator-ui`（前者为 Mojo 薄封装，后者为前端，无后端风险面）。
